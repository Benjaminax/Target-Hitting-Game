# Target-Hitting-Game

A fast-paced target practice game built in Unreal Engine 5! Inspired by Call of Duty, this game features target collision, a timer, hit counter, breakable gun mechanics, and a scoreboard in a detailed "Village" environment. Aim, shoot, and hit your high score!

## Features

- **Target collision:** Realistic hit detection and instant feedback when you hit a target.
- **Timer & hit counter:** Race against a countdown timer while tracking your accuracy and hits.
- **Breakable gun mechanics:** Weapons degrade with use and can break, forcing you to adapt.
- **Scoreboard:** Persistent high-score tracking to compete against your best performance.
- **Village environment:** A detailed, immersive "Village" map designed for intense target practice.

## Getting started

1. Clone the repository to your local machine.
2. Open the project in Unreal Engine 5.3 or later.
3. Ensure the required engine plugins (Chaos Physics, Enhanced Input) are enabled.
4. Press Play in the editor or package the game for your target platform.
5. No API keys or external setup required — just launch and play.

## Dependencies

- **Unreal Engine 5:** Built entirely with UE5's native systems.
- **Chaos Physics:** For realistic target collision and gun breakage physics.
- **Enhanced Input System:** For smooth, responsive aiming and shooting controls.
- **Niagara VFX:** For impact effects and visual feedback.
